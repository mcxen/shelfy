About Shelfy

Your downloads, tamed. A silent, local-first file organizer built for people who value privacy and peace of mind.

The Story

Why Shelfy exists.

Origin Story

I built Shelfy because, like many people, I found my Downloads folder had become a digital wasteland. After one too many frantic searches for a PDF buried under a pile of memes, installers, and random screenshots, I realized something had to change.

Existing tools were either bloated, subscription-based, or sent my file names to someone's cloud. I needed something that just works in the background, never phones home, and stays out of my way.

Project Goals

Shelfy does not try to be the most advanced file manager in the world. Its goal is to be the simplest and most peaceful tool for organizing your Downloads folder.

  • Zero configuration to get started
  • Works fully offline
  • No accounts, no subscriptions, no ads
  • Respects your attention - no pop-ups

Philosophy

100% local and private. No data ever leaves your computer. Your file names, your rules, your history - everything stays in a local SQLite database. System files like desktop.ini, Thumbs.db, and .DS_Store are never touched.

Open source (MIT). The code is public so anyone can inspect it, modify it, and build on it. There is nothing to hide.

Zero interruptions. Shelfy lives quietly in your system tray. It only speaks up with a subtle toast when it organizes something.

Under the Hood

Shelfy is built with modern, reliable tech:

  • Rust backend - file watcher, SQLite, rule engine
  • Tauri 2.x - lightweight native app shell
  • React 19 + TypeScript - fast, type-safe UI
  • Tailwind CSS - clean, maintainable styling

The entire app weighs ~3.3 MB and uses minimal RAM. You will not notice it is there.

Getting Started

Windows: Download the auto-installer (.exe) or MSI package from the homepage, then launch Shelfy. It starts automatically and places an icon in your system tray.

Linux: Download the AppImage, .deb, or .rpm package from the homepage. See the Linux installation section below for details.

1. Launch Shelfy. On first run, it watches your Downloads folder by default.

2. Left-click the tray icon to open the popup - see recent actions, weekly stats, and clean manually.

3. Right-click the tray icon for the menu: Clean Now, Settings, Quit.

4. Drop a file into your Downloads folder and watch it disappear into the right subfolder within 2 seconds.

Linux Installation

Shelfy runs on most modern Linux distributions. Make sure these libraries are installed (they come pre-installed on most desktop distros):

  • libwebkit2gtk-4.1
  • libayatana-appindicator3

AppImage (universal, no install needed):

chmod +x Shelfy_0.1.2_amd64.AppImage
./Shelfy_0.1.2_amd64.AppImage

.deb (Debian, Ubuntu, Mint, Pop!_OS):

sudo dpkg -i Shelfy_0.1.2_amd64.deb
sudo apt-get install -f

.rpm (Fedora, openSUSE, RHEL):

sudo dnf install Shelfy-0.1.2-1.x86_64.rpm

If your distro uses AppArmor or similar sandboxing, you may need to allow Shelfy to access your home directory so it can watch folders and move files.

Features

Silent by Design - Runs quietly in the background. No pop-ups, no interruptions - just a subtle toast when files are organized.

Smart Rules Engine - Define custom rules with extensions, regex patterns, and dynamic paths like {year}/{month}.

History & Undo - Every action is logged locally. Made a mistake? Undo any move with a single click. You can also Revert All from the History tab.

7 Languages - Auto-detects your system language. English, Polish, Italian, German, French, Russian, and Japanese supported.

Dark Mode - Follows your Windows theme or switch manually between Light, Dark, and System.

Autostart - Shelfy registers itself to start with Windows automatically. Toggle this in Settings → General.

Manual & Paused Folder Modes - Set a folder to collect files without auto-moving, or pause watching entirely.

Scheduled Cleaning - Run Clean Now automatically up to four times per day at fixed times.

Extended Grace Period - Delay file moves from seconds up to 48+ hours.

Rule Export / Import - Back up and share your rules as JSON.

Configuration

Open Settings from the tray menu to customize Shelfy.

Watched Folders - Add or remove folders that Shelfy monitors. By default, only your Downloads folder is watched.

Rules - Create rules based on file extensions, regex patterns, or wildcards. Set custom destination folders with placeholders:

  • {year} - current year (e.g. 2026)
  • {month} - current month (e.g. 05)
  • {filename} - original file name without extension

General - Change language, theme, and autostart behavior.

Ignore Rules

Shelfy can skip specific files or folders using ignore rules. You can set them up in Settings → Ignore, or create a .shelfyignore file manually in any watched folder.

How it works. When Shelfy detects a new file, it first checks the .shelfyignore file in that folder. If the file name matches any rule, Shelfy leaves it alone. Simple as that.

Adding rules in the app. Open Settings → Ignore, select a watched folder, and type your pattern in the input field. Press Enter or click Add, then Save. Shelfy will create (or update) the .shelfyignore file automatically.

Writing rules manually. Create a plain text file named .shelfyignore in the folder you want to control. Write one rule per line. Lines starting with # are comments and ignored.

Rule types:

  • Exact name — desktop.ini ignores only that exact file.
  • Wildcard — *.tmp ignores all files ending with .tmp. ~$* ignores all Office temp files.
  • Folder — node_modules/ (with a trailing slash) ignores that folder entirely.

Common examples:

# Ignore Office temp files
~$*

# Ignore all .tmp files
*.tmp

# Ignore the node_modules folder
node_modules/

# Ignore macOS system files
.DS_Store

# Ignore Windows thumbnails
Thumbs.db

Tips:

  • Rules are case-insensitive on Windows.
  • Each watched folder can have its own .shelfyignore file.
  • If you delete the .shelfyignore file, all files in that folder become eligible for sorting again.
  • System files like desktop.ini are already ignored by default — you do not need to add them.

Privacy & Security

Shelfy is built on a local-first philosophy. This means:

  • Zero cloud connections
  • Zero telemetry or analytics
  • All data stored in a local SQLite database
  • No accounts or registration required

VirusTotal - You can verify Shelfy's safety on VirusTotal. Any detections are false positives from heuristic engines due to the app being new and unsigned.

SmartScreen - Windows may show a warning because Shelfy is not signed with a code certificate (which costs ~$400/year). Click "More info" → "Run anyway" to install.

FAQ

How does Shelfy work?
Shelfy sits in your system tray and watches selected folders. When a new file appears, it waits 2 seconds to ensure the download is complete, then moves it based on your rules.

Is my data sent anywhere?
No. Shelfy is 100% offline. Nothing ever leaves your computer.

Can I customize the rules?
Absolutely. Open Settings → Rules to create custom sorting logic.

What if Shelfy moves a file to the wrong place?
Open Settings → History. Every action can be undone with one click.

Does it start automatically with Windows?
Yes. You can toggle this in Settings → General.

Is it really free?
Yes. MIT License. Use it, modify it, build your own version.

Troubleshooting

Shelfy does not sort files.
Make sure the folder is added to Watched Folders and that at least one rule matches the file extension.

Undo does not work.
The destination file must still exist. If you moved or deleted it manually, undo cannot restore it.

Windows SmartScreen blocks the installer.
This is expected for unsigned apps. Click "More info" → "Run anyway".

High CPU or RAM usage.
Shelfy is designed to be lightweight. If you experience issues, check your Watched Folders - monitoring network drives or folders with thousands of files can cause load.
