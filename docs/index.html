把杂乱文件夹整理成可维护的工作流。

Shelfy 是桌面端文件自动化工具。它可以整理下载目录、备份文件夹、定时执行规则、按文本内容识别文件，也可以自动解压和压缩归档文件。

核心能力

不是“清理软件”，而是文件规则引擎。

Shelfy 保持简单入口：托盘、设置页、快捷任务列表。复杂规则交给 Orden 引擎，执行结果有日志，运行前可以先模拟。

整理文件

按扩展名、文件名、正则、大小、时间、哈希、重复文件、MIME、EXIF 等条件筛选并移动、复制或重命名。

备份文件夹

把文档、项目或下载内容复制到一个或多个目标位置。保留原件，适合轻量备份和归档。

定时执行

支持固定时间和 cron 表达式，后台 keepalive 与调度日志帮助你确认任务是否执行。

可视化规则，也可以直接写 YAML。

常见任务用 Visual 模式配置；需要精细控制时切到 Source 模式，直接编辑 Orden YAML。

运行前先模拟。

Check / Simulate / Run 分离。模拟结果会列出命中的文件和即将执行的动作，降低误操作风险。

rules:
  - name: 备份 PDF 文档
    locations:
      - ~/Downloads
    subfolders: true
    filters:
      - extension: [pdf]
    actions:
      - copy: ~/Documents/Shelfy Backups/PDF/

压缩包自动化

下载完压缩包，规则可以接着处理。

Orden 支持 extract / compress 动作。ZIP 内置处理；7z 和 rar 可通过系统 7-Zip 处理。支持密码列表、默认压缩密码和成功后删除原文件。

自动解压

匹配 zip、7z、rar 后解压到指定目录。可配置多个密码并自动尝试。

自动压缩

把文件或文件夹压缩成 zip / 7z。ZIP 支持 AES-256 密码加密。

执行日志

每个创建目录、解压文件、跳过冲突、删除原包的动作都会进入执行日志。

文本内容识别

不只看文件名，也能看文件内容。

filecontent 过滤器支持 txt、markdown、日志、代码、配置文件、Dockerfile、Makefile、PDF 和 DOCX。适合识别发票号、关键字、TODO、项目配置或日志内容。 

rules:
  - name: 收集包含 TODO 的代码
    locations:
      - ~/Downloads
    subfolders: true
    filters:
      - filecontent: "TODO|FIXME"
    actions:
      - move: ~/Documents/Code Notes/

使用方式

从简单规则开始，逐步加自动化。

不需要一次配置完所有东西。先选择一个文件夹，再添加规则，模拟确认后再运行。

选择文件夹

添加 Downloads 或任何需要整理、备份、监控的目录。

配置规则

用简单规则或 Orden Visual 模式描述筛选条件和动作。

先模拟

查看命中文件、目标路径和动作日志，确认无误后再执行。

定时运行

需要自动化时启用调度，让规则按时间或 cron 执行。
